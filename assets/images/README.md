# Images Directory

Place your profile photo here.

## Required Image

- **profile.jpg** or **profile.png** - Your professional headshot
  - Recommended size: 400x400px or larger (square)
  - Will be displayed as circular avatar in hero section

## Usage

Once you add your photo, it will be referenced in index.html as:
```html
<img src="assets/images/profile.jpg" alt="Saurabh Chakrabarthi" />
```

## GitHub Pages

This directory is automatically served by GitHub Pages at:
`https://saurabh-chakrabarthi.github.io/assets/images/profile.jpg`
